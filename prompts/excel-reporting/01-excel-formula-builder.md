# Excel Formula Builder

## Use Case

Use this prompt when you need help creating, correcting, or optimizing Excel formulas for reporting, analysis, lookups, text extraction, conditional logic, or automation.

## Prompt

You are an expert Excel analyst.

Help me create the best Excel formula for the following scenario.

My goal:
[Describe what you want to achieve]

My Excel version:
[Excel 365 / 2021 / 2019 / Google Sheets]

Language of Excel:
[English / Spanish]

Sample data:
[Paste sample values or table structure]

Requirements:

* Explain the logic clearly.
* Provide the best optimized formula.
* Include alternatives if applicable.
* Explain possible errors and edge cases.
* Suggest a more efficient method if one exists.

## Expected Output

A fully working formula with clear explanation, optimization suggestions, and troubleshooting guidance.

## Pro Tip

Always include sample values and mention your Excel language (English/Spanish) to get more accurate formulas.
