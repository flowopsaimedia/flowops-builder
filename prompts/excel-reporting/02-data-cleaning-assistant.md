# Data Cleaning Assistant

## Use Case

Use this prompt when you need to clean, standardize, validate, or prepare messy Excel or spreadsheet data for reporting, dashboards, or analysis.

## Prompt

You are an expert Excel data cleaning specialist.

Help me clean and standardize the following dataset.

My goal:
[Describe what the clean data should look like]

Data issues detected:
[Duplicates / Empty values / Wrong formats / Mixed languages / Typos / Inconsistent values]

Sample data:
[Paste sample rows or structure]

Requirements:

* Identify all data quality issues.
* Suggest cleaning strategies.
* Provide Excel formulas when needed.
* Recommend Power Query or automation if applicable.
* Explain the safest and most scalable approach.

## Expected Output

A step-by-step data cleaning strategy, formulas, and optimization recommendations.

## Pro Tip

Paste 5–10 representative rows of messy data to get much more accurate cleaning recommendations.
