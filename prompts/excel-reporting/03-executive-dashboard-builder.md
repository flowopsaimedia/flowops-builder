# Executive Dashboard Builder

## Use Case

Use this prompt when you need to design executive dashboards in Excel for KPIs, business reporting, project tracking, or operational visibility.

## Prompt

You are an expert Excel dashboard consultant.

Help me design a professional executive dashboard.

Dashboard goal:
[What should leadership understand quickly?]

Audience:
[Executives / Managers / Operations / Technical teams]

Metrics/KPIs:
[List the indicators]

Data source:
[Excel sheets / Manual reporting / Database export]

Requirements:

* Recommend the best dashboard structure.
* Suggest chart types and layouts.
* Explain KPI prioritization.
* Recommend colors and readability practices.
* Suggest automation opportunities.

## Expected Output

A complete dashboard blueprint with KPI recommendations, layout guidance, and reporting best practices.

## Pro Tip

Always explain who the audience is. Executive dashboards and operational dashboards should look very different.
