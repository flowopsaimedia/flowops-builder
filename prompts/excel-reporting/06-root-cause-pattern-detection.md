# Root Cause Pattern Detection

## Use Case

Use this prompt when recurring issues, incidents, defects, or failures appear in data and you need to identify patterns and probable root causes.

## Prompt

You are an expert root cause analyst.

Help me identify recurring patterns and possible root causes in the following data.

Problem description:
[Describe the issue]

Observed symptoms:
[List symptoms]

Historical data:
[Paste data or examples]

Frequency:
[How often does it happen?]

Requirements:

* Detect recurring patterns.
* Identify possible root causes.
* Suggest hypotheses.
* Recommend validation methods.
* Prioritize probable causes.

## Expected Output

A root cause analysis with prioritized hypotheses and actionable recommendations.

## Pro Tip

Provide timestamps, categories, or recurring conditions to improve pattern detection accuracy.
