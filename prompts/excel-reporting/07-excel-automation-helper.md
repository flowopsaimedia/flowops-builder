# Excel Automation Helper

## Use Case

Use this prompt when repetitive Excel work consumes too much time and automation opportunities are needed.

## Prompt

You are an Excel automation expert.

Help me automate the following repetitive Excel process.

Current process:
[Describe the manual process]

Pain points:
[What's taking too much time?]

Frequency:
[Daily / Weekly / Monthly]

Current tools:
[Excel / Power Query / Macros / VBA / Power Automate]

Requirements:

* Identify automation opportunities.
* Suggest the best automation method.
* Recommend formulas, Power Query, VBA, or other tools.
* Prioritize simplicity and maintainability.
* Estimate time savings.

## Expected Output

A practical automation strategy with recommended tools and implementation steps.

## Pro Tip

Describe the exact repetitive steps instead of the general goal for better automation recommendations.
