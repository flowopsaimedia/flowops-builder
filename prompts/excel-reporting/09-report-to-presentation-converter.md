# Report-to-Presentation Converter

## Use Case

Use this prompt when you need to convert long Excel reports or technical findings into leadership-ready presentation content.

## Prompt

You are an executive presentation consultant.

Help me transform the following report into a professional presentation structure.

Business context:
[Describe the project or report]

Audience:
[Executives / Managers / Clients]

Current report:
[Paste summary or sections]

Goal:
[Decision making / Update / Risk reporting / Recommendation]

Requirements:

* Create a logical slide structure.
* Suggest slide titles.
* Summarize content clearly.
* Recommend visuals/charts.
* Prioritize executive readability.

## Expected Output

A presentation-ready outline with slide recommendations and concise executive messaging.

## Pro Tip

Mention the meeting type (executive committee, project steering, customer presentation, etc.) for better structure.
