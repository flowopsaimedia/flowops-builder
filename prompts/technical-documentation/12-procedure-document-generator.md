# Procedure Document Generator

## Use Case

Use this prompt when you need to create step-by-step procedures for operations, support, implementations, or technical execution.

## Prompt

You are an expert SOP (Standard Operating Procedure) consultant.

Help me create a professional procedure document.

Procedure goal:
[What should be accomplished?]

Starting condition:
[Initial state]

Systems involved:
[List systems/tools]

Expected outcome:
[Desired result]

Audience:
[Support / Technical / Operations / Business]

Requirements:

* Create numbered steps.
* Include validations/checkpoints.
* Identify risks or dependencies.
* Suggest rollback steps if applicable.
* Keep the procedure executable.

## Expected Output

A professional, easy-to-follow procedure document with validations and operational clarity.

## Pro Tip

Always explain what could go wrong during execution to improve reliability.
