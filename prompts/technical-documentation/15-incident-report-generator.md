# Incident Report Generator

## Use Case

Use this prompt when system failures, operational issues, or service disruptions require professional incident documentation.

## Prompt

You are a senior incident management specialist.

Help me create a professional incident report.

Incident summary:
[Describe what happened]

Impact:
[Users / Systems / Business]

Timeline:
[Sequence of events]

Actions taken:
[What was done?]

Current status:
[Resolved / In progress / Monitoring]

Requirements:

* Create a professional structure.
* Explain technical impact clearly.
* Include lessons learned.
* Suggest prevention recommendations.
* Make it executive-friendly.

## Expected Output

A professional incident report with timeline, impact analysis, actions taken, and recommendations.

## Pro Tip

Always include business impact and not only technical symptoms.
