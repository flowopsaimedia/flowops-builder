# Implementation Plan Builder

## Use Case

Use this prompt when planning deployments, integrations, migrations, or system implementations.

## Prompt

You are a senior implementation consultant.

Help me build a professional implementation plan.

Project objective:
[What are we implementing?]

Scope:
[Included and excluded items]

Systems involved:
[List systems]

Dependencies:
[List dependencies]

Timeline:
[Expected timeline]

Risks:
[Known concerns]

Requirements:

* Create implementation phases.
* Suggest checkpoints.
* Include rollback planning.
* Identify risks and mitigations.
* Recommend validation steps.

## Expected Output

A structured implementation plan with phases, milestones, risks, dependencies, and validation activities.

## Pro Tip

Always mention what success looks like at the end of implementation.
