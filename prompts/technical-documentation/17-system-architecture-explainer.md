# System Architecture Explainer

## Use Case

Use this prompt when technical systems or architectures need to be explained clearly for documentation or stakeholder understanding.

## Prompt

You are a senior solution architect.

Help me explain the following system architecture clearly.

System objective:
[What does the system do?]

Components:
[List systems/services]

Integrations:
[List interactions]

Audience:
[Technical / Business / Executives]

Requirements:

* Explain architecture clearly.
* Simplify complex concepts.
* Suggest diagrams if applicable.
* Identify dependencies.
* Highlight risks or bottlenecks.

## Expected Output

A professional architecture explanation adapted to the audience level.

## Pro Tip

Explain whether the audience is technical or executive to control complexity.
