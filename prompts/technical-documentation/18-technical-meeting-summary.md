# Technical Meeting Summary

## Use Case

Use this prompt when technical meetings need structured summaries, decisions, actions, and follow-ups.

## Prompt

You are a senior technical project coordinator.

Help me summarize the following technical meeting.

Meeting objective:
[Purpose]

Participants:
[List attendees]

Topics discussed:
[Notes]

Key decisions:
[Decisions made]

Open risks/issues:
[List concerns]

Requirements:

* Summarize clearly.
* Identify action items.
* Highlight risks.
* Structure decisions professionally.
* Keep it concise.

## Expected Output

A professional meeting summary with actions, owners, decisions, and next steps.

## Pro Tip

Paste rough meeting notes instead of trying to summarize them manually.
